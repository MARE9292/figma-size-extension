# Figma 尺寸延展规范（Tiger Size Extension）

把**一张源设计稿**延展成标准投放尺寸 × 三语言，保证团队产出**规范一致**。这是硬规范，按它执行，不要自由发挥版面。

## 0. 触发后先做
1. 读源稿：`get_metadata` → `get_design_context` → 截图，看清内容、字号、图层、图片填充（记下 imageHash 以便复用）。
2. 判定场景：源稿属于「活动中心 Banner」类（宽幅 文字+主视觉）还是「弹窗」类（竖版促销）。
3. 跟用户确认：目标尺寸（默认两种都做）、语言（默认英/简/繁）、英繁文案是否由你翻译（默认是，交付前请市场复核）。
4. 严格执行下面的「版面不变量」。逐帧用截图自检。

## 1. 标准尺寸 × 语言
| 场景 | 尺寸(px) | 语言 |
|---|---|---|
| 活动中心 Banner | **1035 × 368**（宽） | 简 / 繁 / 英 |
| 弹窗 Popup | **900 × 1200**（竖） | 简 / 繁 / 英 |

每个尺寸出 3 个语言帧，命名：`<活动名>-活动中心-1035x368-{简略空/繁/EN}`、`<活动名>-弹窗-900x1200-{/繁/EN}`。

## 2. 字体（默认）
- 简体中文 → **Source Han Sans CN**（思源黑体简）
- 繁体中文 → **Source Han Sans HK**（思源黑体繁/港；注意 TC 在云端不可用，繁体一律用 HK）
- 英文 → **Roboto**
- 字重：主标题 **Heavy**、副标题 **Medium**、正文/合规 **Regular**、标签/按钮 **Bold**
- **行高**：主标题、副标题一律 **130%**（lineHeight PERCENT 130）。
- **主副间距**：主标题与副标题之间垂直间距固定 **20px**（副标题 top = 主标题 bottom + 20）。
- **3 行例外**：当文案为 **3 个单独上下排列的文本元素**时（如 主标题 / 产品字 / 副标题），三者之间行间距 = **0px**（自动布局 `itemSpacing=0`），覆盖上面的 20px。注意：仅 2 个文本、其中某个**自身折行**的情况**不算**"3 行"，仍按 20px。以上适用所有尺寸、所有物料。
- 思源 CN/HK 在云端 `use_figma` 可直接 `loadFontAsync` 应用，**默认无需任何插件**。
- 仅当用户**明确要求苹方 / MiSans** 时，才用本 skill 附带的本地字体插件落地（见 `references/font-swap/`）。

## 3. 版面不变量（硬规则）

### 活动中心 Banner 1035×368
- 布局：左文字 + 右主视觉安全区；四周固定边距，延展时边距不缩放。
- **文案（主标题+副标题）距顶部恒 105px**（即最上方文案 `y=105`）。
- 标签 ≥30px；主标题 ≥45px 加粗；副标题 ≥36px 且 **≤1 行**；主+副 ≤3 行。
- **永远不要加合规文字。**
- **右侧主视觉固定在右侧 410px 宽区域内**（区域 x ∈ [625, 1035]，即左侧文案区固定 625px）。主视觉等比缩放使其宽 ≤410，整组放入该区域，垂直居中；**不越出该区、不裁切、不压左侧文字**。

### 弹窗 Popup 900×1200
- 容器圆角 **20px**，导出 **PNG**。广告角标系统自带、不手工加。
- 结构（从上到下）：角标 → 主标题 → 副标题 → 核心视觉安全区 → CTA 按钮 → 合规。
- **主标题 / 副标题 / 合规 / CTA 按钮 永远水平居中。**
- **永远要有 CTA 按钮**（源稿没有也补）：胶囊形，高 **120**、圆角 **60**、字号 **60**，按文案 hug + 左右内边距各 **72**、水平居中；配色随活动主题（如深底 #252933 黄字 #FFEB33，或金底深棕字）。
- 合规文案 **14px**、置底、居中。
- 主视觉照片**整体等比缩放、不裁切**，居中放核心视觉安全区。

## 4. 高效做法
- 以源稿为母版：**克隆**源节点复用主视觉、产品金属字等（不要重做视觉）。
- 先做「某尺寸 + 简体」一帧调好，再**克隆出繁/英**，仅改文案 + 切字体，三语言版面保持一致。
- 只做尺寸/排版适配，**不改视觉风格、不越安全区**。
- 文案过长（多见英文）：缩短副标题保证 ≤1 行 / 不溢出主视觉。

## 5. 自检清单（交付前逐项核对）
- [ ] 尺寸正确（1035×368 / 900×1200），三语言齐全
- [ ] 字体：简思源CN / 繁思源HK / 英Roboto，字重对
- [ ] Banner：文案顶部 y=105；无合规；副标题单行；照片未裁切
- [ ] 弹窗：主/副/合规/按钮居中；有 CTA（120/60/60/72）；圆角20
- [ ] 三语言版面一致；元素不越安全区；逐帧截图确认无裁切/重叠
